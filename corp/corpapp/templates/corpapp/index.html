Hi there. Are you curious about some data about COVID-19? Same. We gathered some links and some data from around the web, concerning the reactions of countries, states, universities, and companies in Switzerland, Germany, and Switzerland. Have a look around, maybe you find something helpful. Just remember to always double-check your sources, keep calm, and make informed choices.

This project was created during the CodeVsCOVID19 event. The dataset was gathered manually by searching the web and filling out json tables. The backend was realized in Python using the flexible and powerful Django framework. The frontend design was made using Bootstrap, Less, Datatables, and Frappe Charts.
